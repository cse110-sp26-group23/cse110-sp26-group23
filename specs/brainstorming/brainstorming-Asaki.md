# Asaki Billawala - Brainstorming Document

## Concept
Basketball themed typing game.

## Target Audience
CS students and programmers who are intereseted in practicing typing, learning syntax and have an interest in basketball.

## Core Loop
A line of code appears when the user's character is dribbling the ball. The user must type the syntax accurately and quickly. With each success the character moves closer to the hoop. Upon completing a full block of code, the player shoots and dunks the ball. In order for a player to defend themselves there is occasional syntax errors or bugs that appear on the screen, the user must type them quickly to block the shots made by the opposing team.
Dribble: A snippet of code appears at the bottom of the screen.

## The 3 Key Features
- Shot Clock for rewarding timing and accuracy
- Bugs/Syntax errors system for defense
- Unlock different courts/play different teams

## Unique Selling Point
It is an educational tool that uses the interest in basketball and games to teach.

## Inspiration / References
- TypeRacer
- Monkeytype
- Typing Hoops

## Scope
**MVP:** What's the absolute minimum version that's playable/usable?
A typing game that allows a user to type code and shoot hoops when successful.

## Open Questions
- How should we generate the code?
- How long should the code snippet be?
- How do we determine the difficulty of a code snippet and how does the effect scoring?
