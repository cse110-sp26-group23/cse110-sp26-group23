# Crystal Nguyen  - Brainstorming Document

## Concept
A typing game that has a basketball theme. 

## Target Audience
This game is designed for intermediate programmers who wants a fun way to practice and improve their skills. 

## Core Loop
- A code snippet is displayed on the screen
- Player must complete code snippet within a certain time 
- As user continues to accurately type, the basketballs will contiue to sucessfully pass through the hoop. 
- Typing mistkaes would result in the  oponent scoring.
- Playerfs would keep playing to beat their personal record and unlock new levels. 

## The 3 Key Features
- The rewards and scoring system
- The basketball game performance
- Differnt levels/types of challanges

## Unique Selling Point
Combines an arcade style basketball game along with educational purposes.
