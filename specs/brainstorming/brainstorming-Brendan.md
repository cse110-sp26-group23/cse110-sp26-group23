# Brendan Barber - Brainstorming Document

## Concept
A basketball 3-point challenge typing game.

## Target Audience
Programmers who want to learn syntax and practice typing, but want a fun game component.

## Core Loop
The game loop is:
1. Presented with a snippet of text to type
2. Type the text correctly and quickly
3. Play a "Quick-Time Event" to aim your shot
4. Take the shot and get points by scoring
5. Repeat until your run out of time

## The 3 Key Features
- Typing system
- Basketball quick-time-event
- Scoring and high-score

## Unique Selling Point
It is a typing game with a programming focus, and it has a minigame to go along with it

## Inspiration / References
- Wii-Sports Resort 3-Point Challenge
- Type racer
- Monkey type

## Scope
**MVP:** What's the absolute minimum version that's playable/usable?

The minimum version would be a simple typing interface, a quick-time-event, and scoring system.

## Open Questions
Things that need to be figured out before or during development.
- How do you visualize an interesting quick-time-event game?
- Would we want a public leaderboard? Your score gets sent off somewhere?
