# Nhan Doan - Brainstorming Document

## Concept
A basketball-themed typing game where players type words correctly to shoot, score, and build a streak.

## Target Audience
Students and casual players who want to practice typing in a fun way. The game would be playable on desktop browsers.

## Core Loop
The player sees a word, types it correctly, and earns a basketball shot. Correct typing increases the score and streak, while mistakes reduce accuracy or break the combo.

## The 3 Key Features
- Typing words to shoot basketballs
- Score, timer, and accuracy tracking
- Combo/streak system for consecutive correct words

## Unique Selling Point
It combines typing practice with basketball arcade mechanics, making typing feel more like a sports challenge instead of a normal typing test.

## Inspiration / References
Typing games like Monkeytype and Nitro Type for speed/accuracy practice, plus basketball arcade games for the scoring and streak system.

## Scope
**MVP:** A simple browser game where a word appears, the player types it, and the game tracks score, time, accuracy, and correct/incorrect answers.

## Open Questions
- Should the game focus more on speed, accuracy, or both?
- Should difficulty increase over time?
- Should we add animations and sound effects in the first version or save them for later?
