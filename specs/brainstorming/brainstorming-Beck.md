# Beckham Yeoh - Brainstorming Document

## Concept
Basketball-themed typing game

## Target Audience
Aspiring programmers learning code syntax

## Core Loop
Code is displayed
User types code
Animation plays
Continue until timer is up

## The 3 Key Features
- User can choose to pass/shoot/dribble with possession, and intercept/block/steal when defending
- Actions have higher success rate when code is typed quickly and correctly
- Play against different teams (Claude warriors, Java Castle, etc) to train on different code

## Unique Selling Point
Get more immersed in basketball matches while learning code

## Inspiration / References
Keys of Fury

## Scope
A simple typing game with different coding languages

## Open Questions
Is the AI able to generate decent visuals?

## User story
As a CS student cramming for a Java midterm, I want to play a basketball match where I shoot, pass, and dribble by typing Java code snippets against themed opponents like Java Castle, so that I can drill syntax into muscle memory through an engaging game instead of burning out on flashcards.
